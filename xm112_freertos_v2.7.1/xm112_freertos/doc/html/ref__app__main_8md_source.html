3 Reference Applications are applications written to showcase how a use-case can be achieved by using Acconeer Detectors and/or Services. They are written for a specific use-case setup and are guaranteed to achieve proper functionality for that setup. The purpose is for the reference applications to be a good starting point for product development within the area of the use-case.
